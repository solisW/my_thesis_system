# 配置说明

系统配置集中在 `src/config.py`，运行时优先读取环境变量。批处理启动脚本会提供一组默认值，实际部署时建议通过系统环境变量或进程管理工具覆盖。

## 数据库配置

| 变量 | 默认值 | 说明 |
| --- | --- | --- |
| `DATABASE_URL` | 空 | 若设置则直接作为 SQLAlchemy URI 使用 |
| `DB_BACKEND` | `mysql` | `mysql` 或其他值；非 `mysql` 时使用 SQLite |
| `MYSQL_HOST` | `127.0.0.1` | MySQL 地址 |
| `MYSQL_PORT` | `3306` | MySQL 端口 |
| `MYSQL_DATABASE` | `gas_monitor` | 数据库名 |
| `MYSQL_USER` | `root` | 数据库用户 |
| `MYSQL_PASSWORD` | 本机启动脚本默认值 | 数据库密码，部署时必须通过环境变量覆盖 |

SQLite 本地库路径：

```text
instance/gas_monitor.db
```

## Flask 与安全配置

| 变量 | 默认值 | 说明 |
| --- | --- | --- |
| `FLASK_HOST` | `127.0.0.1` | Web 服务监听地址 |
| `FLASK_PORT` | `5000` | Web 服务端口 |
| `FRONTEND_URL` | `http://127.0.0.1:5173` | 后端页面路由跳转到的前端地址 |
| `FRONTEND_ORIGINS` | `http://127.0.0.1:5173,http://localhost:5173` | 允许访问后端 API 的前端来源白名单 |
| `APP_SECRET_KEY` | `gas-monitor-secret-key` | Flask session 密钥 |
| `APP_CREDENTIAL_SECRET` | `gas-monitor-default-secret` | 用户/工程师敏感字段和系统凭据的字段级加密密钥 |

生产环境必须覆盖 `MYSQL_PASSWORD`、`APP_SECRET_KEY` 和 `APP_CREDENTIAL_SECRET`，不要把真实密码写入文档或公开仓库。

`APP_CREDENTIAL_SECRET` 会派生 Fernet 对称加密密钥，用于加密用户名、姓名、工号、电话、工程师片区、地址和头像等身份数据。生产环境应保持该值稳定并纳入密钥管理；如果需要更换密钥，应先实现密钥轮换脚本，不能直接替换环境变量。

## 模型与训练参数

| 配置 | 默认值 | 说明 |
| --- | --- | --- |
| `WINDOW_SIZE` | `24` | LSTM AutoEncoder 输入窗口长度 |
| `TRAIN_EPOCHS` | `20` | 默认训练轮数 |
| `BATCH_SIZE` | `32` | 批大小 |
| `LEARNING_RATE` | `0.001` | 学习率 |
| `HIDDEN_SIZE` | `32` | LSTM 隐藏层维度 |
| `NUM_LAYERS` | `2` | LSTM 层数 |
| `TRAIN_SPLIT` | `0.8` | 训练集比例 |
| `TRAINING_INTERVAL_SECONDS` | `300` | 持续训练脚本默认间隔 |

模型生成物路径：

```text
models/lstm_autoencoder.pt
models/scaler.pkl
models/train_meta.json
models/active_model.json
models/registry/
```

这些文件是运行生成物，不纳入系统文档管理。

## 在线检测配置

| 变量 | 默认值 | 说明 |
| --- | --- | --- |
| `ASYNC_DETECTION_ENABLED` | `1` | 是否启用异步检测 |
| `ASYNC_DETECTION_WORKERS` | `4` | 异步检测线程数 |
| `ONLINE_TIMEOUT_SECONDS` | `15` | 设备在线超时阈值，代码常量 |
| `ENGINEER_ONLINE_TIMEOUT_SECONDS` | `45` | 工程师在线超时阈值 |
| `PHYSICAL_DATA_RETENTION_DAYS` | `30` | 物理设备历史数据保留天数 |

在线检测遵循论文初稿中的处理链路：按设备缓存连续读数，达到 `WINDOW_SIZE=24` 后提取瞬时流量、累计用气量、电池电压、信号强度、阀门状态、环境温度和管网压力 7 类特征，进入 LSTM AutoEncoder 推理并融合业务规则。

## 实验评估输出

`scripts/evaluate_lstm_autoencoder_experiment_results.py` 默认输出到：

```text
data/experiment_results/
```

该目录用于保存方法对比、ROC、阈值扫描、重构曲线和误差分布等运行生成物，不作为 `docs/` 文档内容维护。

## 配置修改影响

| 修改项 | 需要同步检查 |
| --- | --- |
| 数据库连接 | `start_system.bat`、部署文档、健康检查脚本、本地测试库 |
| `APP_CREDENTIAL_SECRET` | 历史用户和工程师密文字段是否还能解密 |
| `WINDOW_SIZE` 或 `FEATURE_COLUMNS` | 训练数据、模型、scaler、在线检测、重构曲线和实验导出 |
| MQTT 主题 | 设备注册主题、上传主题、外部 Broker 发布配置 |
| `FRONTEND_URL` 或端口 | 浏览器访问地址、后端跳转、跨域来源白名单 |
| 漂移阈值 | 设置页展示、运维巡检和复训判断 |

生产环境修改配置前建议先记录旧值。涉及数据库、加密密钥、模型窗口或特征列的配置变更，应先在本地或测试库验证，再应用到正式环境。
