# 前后端分离说明

系统已新增独立前端工程，前端不再依赖 Flask 模板渲染。

## 技术栈

- 前端：Vue 3、ECharts、Leaflet
- 后端：Flask、Flask-SQLAlchemy、Flask-Sock
- 通信：HTTP JSON API、WebSocket

## 目录

```text
frontend/index.html      独立前端入口
frontend/src/main.js     Vue 页面、状态、API 调用、WebSocket、ECharts、Leaflet
frontend/src/styles.css  前端样式
frontend/package.json    前端工程描述
```

迁移前的 `frontend/templates/` 和 `frontend/static/` 已删除。当前前端只维护独立 Vue 入口和 `frontend/src/` 资源，后端页面路由仅负责重定向到独立前端。

## 启动

后端 API：

```powershell
.\start_system.bat
```

前端：

```powershell
.\start_frontend.bat
```

访问：

```text
前端：http://127.0.0.1:5173
后端：http://127.0.0.1:5000
```

## 接口边界

- 登录：`POST /api/auth/login`
- 会话：`GET /api/auth/session`
- 退出：`POST /api/auth/logout`
- 业务数据：`/api/*`
- 实时推送：`/ws`

前端请求默认携带 Cookie 会话，后端通过 `FRONTEND_ORIGINS` 允许独立前端来源跨域访问。
